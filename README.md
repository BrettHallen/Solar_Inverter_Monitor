# Solar Inverter Monitor
Our solar power system has two 5kW inverters by SAJ Electric with WiFi modules.<br>

I wanted a simple dashboard so I could see at a glance when our system was generating enough power to cover the dishwasher, washing machine, etc.<br>

Implemented for the M5Stack Cardputer for now.<br>
